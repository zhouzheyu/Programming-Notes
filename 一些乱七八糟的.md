## Pycharm操作指南

## 服务器

### 通过SSH连接

首先打开配置文件：

```
nano ~/.ssh/config
```

把这些放到ssh config，粘贴以下内容：

```bash
Host j17
    HostName 210.28.133.134
    User zzy
    ProxyJump js16

Host js16
    HostName 127.0.0.1
    User proxy_server
    Port 36622
    ProxyJump sv01

Host sv01
    HostName 119.45.119.205
    User proxy_server
```

在终端连接服务器：

```bash
ssh j17
```

保存退出：

```
Ctrl + O → 回车
Ctrl + X
```

然后执行：

```
chmod 600 ~/.ssh/config
```

最后在终端输入指令即可连接：

```
ssh j17
```

**如果在PyCharm里面连接？**

* 先把 j17 加成 SSH 主机
* 给本地项目换一个远程解释器
  * 选择：`/data2/zzy/miniconda3/envs/myenv/bin/python`
