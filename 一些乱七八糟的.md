# 一些乱七八糟的

## 虚拟环境

激活虚拟环境：

```bash
conda activate ./env
```

env.yml:

```python
name: cs224n_dfp
channels:
  - defaults
dependencies:
  - python=3.8
  - pip
  - pip:
      - torch
      - torchvision
      - torchaudio
      - tqdm==4.58.0
      - requests==2.25.1
      - importlib-metadata==3.7.0
      - filelock==3.0.12
      - sklearn==0.0
      - tokenizers==0.20
      - explainaboard_client==0.0.7
      - einops==0.8.0
      - transformers==4.46.3
      - sacrebleu==2.5.1
```

启动：

```
conda env create -f env.yml
conda activate cs224n_dfp
```

根据requirments.txt安装包：

```python
pip install -r requirements.txt
```

## Pycharm操作指南

双击```shift```：开启访达

command + 鼠标左键：进入函数内部

选中函数或类名右键 → 选择 **Find Usages** （查找用法）：快速查看函数/类在哪里被调用
